# Next.js Cloudinary Video Lab

A Next.js project for experimenting with Cloudinary video capabilities.

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- Next.js app directory
- Cloudinary video integration
- Video processing and transformation
